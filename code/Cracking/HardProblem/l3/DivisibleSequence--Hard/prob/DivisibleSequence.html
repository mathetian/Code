DivisibleSequence

Single Round Match 565 Round 1 - Division II, Level Three

Statement

A divisible sequence with starting value N and length H is a sequence of positive integers with the following properties: The sequence has H elements, labeled A[0] through A[H-1]. A[0] equals N. For each i between 0 and H-2, inclusive, A[i+1] divides A[i].

You are given the ints N and H. Let C be the count of all divisible sequences with starting value N and length H. Return the value (C modulo 1,000,000,009).

Definitions

  • Class: DivisibleSequence
  • Method: count
  • Parameters: int, int
  • Returns: int
  • Method signature: int count(int N, int H)

Constraints

  • N will be between 1 and 1,000,000,000, inclusive.
  • H will be between 1 and 1,000,000,000, inclusive.

Examples

Example 1

Input

5,
3

Output

3

Reason

The three possible sequences are:

5, 5, 5

5, 5, 1

5, 1, 1

Example 2

Input

6,
3

Output

9

Reason

The nine possible sequences are:

6, 6, 6

6, 6, 3

6, 6, 2

6, 6, 1

6, 3, 3

6, 3, 1

6, 2, 2

6, 2, 1

6, 1, 1

Example 3

Input

10,
2

Output

4

Reason

A[1] can be equal to each of the divisors of the number 10. That is, A[1] can be 1, 2, 5, or 10.

Example 4

Input

1,
10000

Output

1

Reason

The only possible sequence consists of all ones.

Example 5

Input

30,
4

Output

64

Example 6

Input

25883,
100000

Output

991000009
